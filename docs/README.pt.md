# wshm — Agente IA para manutencao de repos GitHub

**Your repo's wish is my command.**

wshm e uma ferramenta CLI + GitHub Action que atua como agente autonomo de manutencao de repositorios.

## Funcionalidades

- **Triagem automatica** — Classificacao, etiquetagem e priorizacao de issues por IA
- **Analise de PRs** — Resumo, avaliacao de risco, checklist de revisao
- **Auto-fix** — Geracao de PRs draft para bugs simples
- **Fila de merge** — Scoring e classificacao de PRs, auto-merge acima do limiar
- **Resolucao de conflitos** — Deteccao e resolucao automatica (nunca force-push)
- **Review inline** — Comentarios de revisao linha por linha por IA
- **Auto-atribuicao** — Atribuicao ponderada de maintainers
- **Blacklist de labels** — Impedir a aplicacao de certos labels
- **Retriagem periodica** — Reavaliacao de triagens obsoletas
- **Notificacoes** — Roadmap diario para Discord, Slack, Teams ou webhook: top 10 issues e PRs pendentes, ordenados por prioridade e antiguidade
- **Dashboard e relatorios** — Dashboard HTML e relatorios Markdown/PDF

Um unico binario. Suas chaves API. Seus dados. Zero infraestrutura.
