# wshm — GitHubリポジトリ管理のためのAIエージェント

**Your repo's wish is my command.**

wshmは、リポジトリの自律的なメンテナンスエージェントとして機能するCLIツール + GitHub Actionです。

## 機能

- **Issueの自動トリアージ** — AIによる分類、ラベリング、優先度設定
- **PR分析** — 要約、リスク評価、レビューチェックリスト
- **自動修正** — シンプルなバグのDraft PR自動生成
- **マージキュー** — PRのスコアリングとランキング、閾値以上で自動マージ
- **コンフリクト解決** — 検出と自動解決（force-pushなし）
- **インラインレビュー** — AIによる行単位のコードレビュー
- **自動アサイン** — 重み付きランダムでメンテナーを割り当て
- **ラベルブラックリスト** — 特定ラベルの適用を防止
- **定期的再トリアージ** — 古いトリアージ結果を定期的に再評価
- **ダッシュボードとレポート** — HTMLダッシュボードとMarkdown/PDFレポート

単一バイナリ。あなたのAPIキー。あなたのデータ。インフラ不要。
